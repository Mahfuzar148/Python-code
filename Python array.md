# üìå Array Slicing in Python

### ‚úÖ Basic Syntax


* **start** ‚Üí ‡¶ï‡ßã‡¶® index ‡¶•‡ßá‡¶ï‡ßá ‡¶∂‡ßÅ‡¶∞‡ßÅ ‡¶ï‡¶∞‡¶¨‡ßá (default = `0`)

* **step** ‚Üí ‡¶ï‡ßü ‡¶ß‡¶æ‡¶™‡ßá ‡¶Ø‡¶æ‡¶¨‡ßá (default = `1`)

üëâ `arr[start:stop:step]` ‡¶∏‡¶¨‡¶∏‡¶Æ‡ßü ‡¶è‡¶ï‡¶ü‡¶æ **‡¶®‡¶§‡ßÅ‡¶® array** return ‡¶ï‡¶∞‡ßá‡•§
