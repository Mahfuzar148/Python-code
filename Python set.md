‡¶è‡¶ï‡¶∏‡¶æ‡¶•‡ßá ‡¶è‡¶ï‡¶æ‡¶ß‡¶ø‡¶ï element ‡¶Ø‡ßã‡¶ó ‡¶ï‡¶∞‡ßá (list, tuple, set ‡¶á‡¶§‡ßç‡¶Ø‡¶æ‡¶¶‡¶ø ‡¶•‡ßá‡¶ï‡ßá)‡•§

## 8. `union()` ‡¶¨‡¶æ `|`

‡¶¶‡ßÅ‡¶ü‡¶ø set ‡¶è‡¶∞ ‡¶∏‡¶¨ element (‡¶°‡ßÅ‡¶™‡ßç‡¶≤‡¶ø‡¶ï‡ßá‡¶ü ‡¶õ‡¶æ‡ßú‡¶æ) return ‡¶ï‡¶∞‡ßá‡•§


## 9. `intersection()` ‡¶¨‡¶æ `&`

‡¶¶‡ßÅ‡¶ü‡¶ø‡¶∞ common element return ‡¶ï‡¶∞‡ßá‡•§


## 10. `difference()` ‡¶¨‡¶æ `-`

‡¶™‡ßç‡¶∞‡¶•‡¶Æ set ‡¶è ‡¶Ü‡¶õ‡ßá ‡¶ï‡¶ø‡¶®‡ßç‡¶§‡ßÅ ‡¶¶‡ßç‡¶¨‡¶ø‡¶§‡ßÄ‡ßü‡¶ü‡¶ø‡¶§‡ßá ‡¶®‡ßá‡¶á‡•§


## 11. `symmetric_difference()` ‡¶¨‡¶æ `^`

‡¶Ø‡ßá element ‡¶¶‡ßÅ‡¶ü‡¶ø set ‡¶è ‡¶Æ‡¶ø‡¶≤‡¶õ‡ßá ‡¶®‡¶æ, ‡¶∂‡ßÅ‡¶ß‡ßÅ ‡¶≠‡¶ø‡¶®‡ßç‡¶®‡¶ó‡ßÅ‡¶≤‡ßã return ‡¶ï‡¶∞‡ßá‡•§

```python
a = {1, 2, 3}
```

## 12. `issubset()`

‡¶è‡¶ï‡¶ü‡¶æ set ‡¶Ö‡¶®‡ßç‡¶Ø‡¶ü‡¶æ‡¶∞ ‡¶Æ‡¶ß‡ßç‡¶Ø‡ßá ‡¶™‡ßÅ‡¶∞‡ßã‡¶™‡ßÅ‡¶∞‡¶ø ‡¶Ü‡¶õ‡ßá ‡¶ï‡¶ø ‡¶®‡¶æ ‡¶ö‡ßá‡¶ï ‡¶ï‡¶∞‡ßá‡•§

```python

b = {1, 2, 3, 4}
print(a.issubset(b))   # True
print(b.issubset(a))   # False
```

---

## 13. `issuperset()`

‡¶è‡¶ï‡¶ü‡¶æ set ‡¶Ö‡¶®‡ßç‡¶Ø‡¶ü‡¶æ‡¶ï‡ßá ‡¶™‡ßÅ‡¶∞‡ßã‡¶™‡ßÅ‡¶∞‡¶ø contain ‡¶ï‡¶∞‡¶õ‡ßá ‡¶ï‡¶ø ‡¶®‡¶æ‡•§

```python

print(b.issuperset(a))   # False
```

## 14. `isdisjoint()`

‡¶¶‡ßÅ‡¶ü‡¶ø set ‡¶è‡¶∞ ‡¶Æ‡¶ß‡ßç‡¶Ø‡ßá ‡¶ï‡ßã‡¶®‡ßã common element ‡¶®‡ßá‡¶á ‡¶ï‡¶ø ‡¶®‡¶æ‡•§

