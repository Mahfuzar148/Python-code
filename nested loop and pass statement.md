---

### Example 4: Pattern printing

```python

    for j in range(i):
        print("*", end="")
    print()
```

**Output:**

```

**
***
****
*****
```

### Example 5: Nested loop with list of list

```python

    for val in row:
        print(val, end=" ")
    print()
```
